## ⚠️ محدودیت‌ها

### امنیت وابسته به:

1. **رمز عبور کاربر:**
   - کاربر باید رمز قوی انتخاب کند
   - حداقل ۱۲ کاراکتر توصیه می‌شود
   - استفاده از ترکیب حروف، اعداد و نمادها

2. **دستگاه کاربر:**
   - دستگاه باید عاری از بدافزار باشد
   - مرورگر باید Web Crypto API را پشتیبانی کند
   - توصیه: استفاده از حالت Incognito/Private

3. **شبکه:**
   - اپلیکیشن کاملاً کلاینت‌ساید است
   - هیچ داده‌ای به سرور ارسال نمی‌شود
   - امنیت شبکه در حوزه کاربر است

## 🚨 عدم مسئولیت

**CryptoMsg** یک ابزار رمزنگاری است، نه یک راه‌حل امنیتی کامل. مسئولیت‌های زیر بر عهده کاربر است:

1. انتخاب رمز عبور قوی
2. محافظت از دستگاه
3. استفاده از HTTPS
4. عدم استفاده در شبکه‌های ناامن
5. پاک کردن حافظه کلیپ‌بورد
